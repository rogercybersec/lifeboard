Gmail Auto-Detection

Auto-scan your Gmail for bills. A script runs in YOUR Google account — emails never leave Google. Only extracted bill data is sent to LifeBoard.

Setup instructions
  1. Choose a shared secret above and save it
  2. Add the same secret as LIFEBOARD_EMAIL_SECRET on Vercel
  3. Click "Copy Gmail Script" above
  4. Go to script.google.com → New project
  5. Paste the script, update YOUR_SECRET and YOUR_LIFEBOARD_URL
  6. Set a trigger: Edit → Triggers → Add → scanForBills → Time-driven → Every hour
  7. Authorise the script when prompted (it only reads your Gmail)
